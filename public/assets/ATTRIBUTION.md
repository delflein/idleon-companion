# Asset attribution

The sprites under this directory (`data/`, `etc/`, `afk_targets/`) are game assets from
*Legends of Idleon* (Lavaflame2 / Codigo Creations), used here under the same informal
fan-tool convention every Idleon fan-site and tool relies on — no explicit asset license,
tacit tolerance from the developer, not original work by this project.

The file list was curated from the [IdleonToolbox](https://github.com/Morta1/IdleonToolbox)
repository's `public/` assets (see `docs/migration/survey-assets.md` for the full audit of
which files are referenced and why). Only the sprite files were copied — none of
IdleonToolbox's own code.
